 O clasă este o structură ce definește caracteristicile și comportamentul unui obiect.
 Clasa este un tip de date definit de utilizator, care poate conține metode, proprietăți și constructori
 pentru a descrie obiectele pe care le poate crea.

O metodă este un bloc de cod care definește comportamentul unei clase.
 Aceasta poate efectua anumite acțiuni și poate returna o valoare sau poate fi de tip void (nu returnează nimic).
 Metodele sunt utilizate pentru a organiza și a executa logica într-un program Java.

O proprietate (sau câmp) este o variabilă definită într-o clasă, care stochează date asociate cu obiectele acelei clase. 
Proprietățile descriu starea unui obiect și sunt accesate și modificate de metodele clasei.

Un constructor este o metodă specială a unei clase care este apelată automat atunci când un obiect este creat. 
Constructorul inițializează starea obiectului, adică valorile proprietăților sale, și 
poate primi parametri pentru a personaliza inițializarea. Un constructor are același nume cu clasa în
 care este definit și nu returnează nimic. Este folosit pentru a asigura că un obiect este într-o stare validă imediat 
după creare.
